# Pomodoro Timer

Simple Pomodoro timer with work/break cycles (CLI or basic GUI).

## Features
- Configurable work/break durations
- Session counter
- (Optional) Sound alert / basic Tkinter GUI

## Tech
- Python 3.x
- Standard library (`time`, `threading`)
- `tkinter` (optional for GUI)

## Run
```bash
python main.py
